GreenOAuth
==========
GreenOAuth
基于OAuth 2.0 统一身份认证授权系统
传统的企业或者一些学校拥有大量的管理系统，但是管理系统之间缺少通信。
基本的身份认证常常不是统一完成而是由各个平台自己完成，一个管理人员可能在不同平台拥有不用的账号密码，难以记忆与管理。
通过GreenOAuth统一身份认证授权系统帮助企业简化管理系统的权限管理。
